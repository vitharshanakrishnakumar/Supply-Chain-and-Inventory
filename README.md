This project is a Power BI dashboard designed to analyze and visualize supply chain and inventory data. The goal is to help organizations monitor stock levels, track product movement, and improve decision-making through interactive dashboards.

The report provides insights into inventory performance, supplier efficiency, and overall supply chain trends.
